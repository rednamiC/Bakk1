During the last few years, there was a major increase in educational satellites launched to the orbit. The start of this was in 1999, when scientists at \acf{Cal Poly} developed the CubeSats.
\newline
Because of the CubeSats being in \acf{LEO}, there is a very small timeframe for the satellites to send data to the \ac{MCC} which leads to a very small data volume \cite{preindl_optimization_2010}. To be able to get more data \acf{GENSO} was devised which connects a network of \acp{GCC} all over the world. Those \acp{GCC} can access the CubeSat of other  teams and transmit the data of the passing satellite \cite{leveque_global_2007}.
\newline
This thesis focuses on the development of a new open source alternative, \ac{BOINSO}, which is build using existing open source projects and the interface of \ac{BOINC} \cite{ries_boinc_2012}.
\newline
\ac{BOINSO} will be released under the \ac{GNU GPL} and is available at \url{https://github.com/Mehnen/BOINSO}.
